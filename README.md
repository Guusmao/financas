# Finanças Pessoais

Aplicativo web local para organizar entradas, saídas, contas fixas, metas e reserva.

## Como usar

Abra `index.html` no navegador.

Os dados ficam salvos no próprio navegador usando `localStorage`. Use os botões de exportar e importar para fazer backup em JSON.

## Estrutura

- `index.html`: interface do app.
- `styles.css`: visual e responsividade.
- `app.js`: cálculos, cadastro, persistência e exportação/importação.
